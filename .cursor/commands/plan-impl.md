# /plan-impl

Create an implementation plan from a specification document.

**Source of truth:** `.cursor/skills/plan-impl/SKILL.md` (workflow is maintained there to avoid duplication).

## Usage

```
/plan-impl docs/specs/{feature-name}.md
```

Or reference a spec file:
```
/plan-impl @docs/specs/my-feature.md
```

## Instructions

When the user invokes `/plan-impl`, follow `.cursor/skills/plan-impl/SKILL.md`.
