# /update-readme

Synchronize README.md with current codebase state.

**Source of truth:** `.cursor/skills/update-readme/SKILL.md` (workflow is maintained there to avoid duplication).

## Usage

Update README based on current repo:
```
/update-readme
```

Update specific sections:
```
/update-readme --section tech-stack
/update-readme --section structure
/update-readme --section status
```

## Instructions

When the user invokes `/update-readme`, follow `.cursor/skills/update-readme/SKILL.md`.
